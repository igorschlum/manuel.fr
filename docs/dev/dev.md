# Développer avec l'IA

Si vous voulez aller plus loin, nous allons aborder comment développer en utilisant l'intelligence artificielle.

## l'IA comme aide

L'IA peut être une aide pour vous apprendre à programmer, on peut lui poser des questions techniques et elle peut vous aider en vous donnant des conseils.

# l'IA en générateur de code

 L'IA peut gérer du code suivant un prompt que vous lui indiquez, elle peut relire du code, le commenter et fabriquer des tests unitaires. C'est-à-dire qu'elle peut aider à vérifier que votre code fonctionne bien.

 On peut aussi lui demander de relire votre code et de le commenter.

# L'IA comme plate-forme de programmation

Il y a un certain nombre de plateforme qui permettent de programmer des interactions entre des systèmes en utilisant l'IA pour le traitement des données.
