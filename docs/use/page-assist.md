# Page-Assist

**Page-Assist** est une extension de navigateur open source qui offre une interface de chat et un panneau latéral pour interagir avec vos modèles Ollama directement depuis votre navigateur.

## Installation

Page-Assist est disponible sur plusieurs navigateurs :

- [**Chrome**](https://chromewebstore.google.com/detail/page-assist-a-web-ui-for/jfgfiigpkhlkbnfnbobbkinehhfdhndo) — 300 000+ utilisateurs, 4,9 ★
- [**Firefox**](https://addons.mozilla.org/en-US/firefox/addon/page-assist/)
- [**Edge**](https://microsoftedge.microsoft.com/addons/detail/page-assist-a-web-ui-fo/ogkogooadflifpmmidmhjedogicnhooa)

Une fois installée, l'extension détecte automatiquement qu'Ollama est lancé et vous permet de choisir le modèle avec lequel vous voulez discuter.

## Fonctionnalités principales

- **Panneau latéral** — Accédez à l'IA depuis n'importe quelle page web (`Ctrl+Shift+Y`)
- **Interface Web** — Interface complète style ChatGPT dans un nouvel onglet (`Ctrl+Shift+L`)
- **Discuter avec la page** — Posez des questions sur le contenu de la page en cours
- **Modèles de vision** — Compatible avec les modèles multimodaux
- **Discuter avec des documents** — Importez et discutez avec des fichiers PDF, CSV, TXT, MD, DOCX
- **Recherche internet** — L'IA peut chercher des réponses sur le web
- **Mémoire** — Conservation du contexte entre les sessions (bêta)

## Fournisseurs supportés

- **Ollama** (principal)
- **Chrome AI / Gemini Nano** (bêta)
- **API compatibles OpenAI** — LM Studio, Llamafile, llama.cpp, vLLM…

## Limites

L'interface est en anglais pour l'instant. Sur Opera et Arc, seul le mode Web UI est disponible (pas de panneau latéral ni de discussion avec la page).
